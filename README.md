# aefi
Source code project manager

<p>Aefi is a CLI program that aims to facilitate the creation of projects in the most different programming languages.
The aim of the project is to avoid the fatigue of always creating the same structure of folders and files, which is boring and repetitive.
The CLI is under development and will soon have its first stable release.
The project is being developed in C/C++, due to its practicality and compatibility with different operating systems.</p>

<h4>Usage:</h4>

```
term@$: <program> <functions> <flags>*
```

<h3>*The flags are optional in some cases.</h3>
